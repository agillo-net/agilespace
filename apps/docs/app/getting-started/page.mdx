# Getting Started with AgileSpace

This guide will help you get up and running with AgileSpace quickly.

## Prerequisites

- Node.js 20 or later
- pnpm 10.4.1 or later

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/agilespace.git
cd agilespace
```

Install dependencies:

```bash
pnpm install
```

## Development

Start the development server:

```bash
pnpm dev
```

This will start all applications in the monorepo in development mode.

## Building for Production

To build all applications for production:

```bash
pnpm build
```
