# Installation

This guide will walk you through the installation process for AgileSpace.

## System Requirements

Before installing AgileSpace, ensure your system meets the following requirements:

- **Node.js**: Version 20 or later
- **Package Manager**: pnpm 10.4.1 or later
- **Operating System**: macOS, Windows, or Linux

## Step 1: Clone the Repository

```bash
git clone https://github.com/yourusername/agilespace.git
cd agilespace
```

## Step 2: Install Dependencies

AgileSpace uses pnpm for package management in a monorepo structure:

```bash
pnpm install
```

## Step 3: Configure Environment Variables

Copy the example environment files:

```bash
cp .env.example .env
```

Edit the `.env` file to configure your environment variables.

## Step 4: Verify Installation

Run the development server to verify everything is working:

```bash
pnpm dev
```

You should now be able to access:
- Dashboard application at [http://localhost:3000](http://localhost:3000)
- Documentation site at [http://localhost:3001](http://localhost:3001)
